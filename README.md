# Text-to-Speech
First ConquerBlocks project using Python.

This first project translates a webpage of interest from text to speech.

The speech must be from the original text of the webpage (otherwise it sounds strange).

This project is missing try-catch blocks and probably many other things I haven't mastered yet, but to be honest, I was impressed when it worked.

If you'd like to make improvements and leave comments explaining your reasoning, I'd be delighted.

Thanks.

# De_Texto_a_Voz
1er trabajo de ConquerBlocks usando python.

Este 1er proyecto locuta de texto a voz una pagina web que te interece.
